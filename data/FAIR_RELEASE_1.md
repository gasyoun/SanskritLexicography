# FAIR Release #1 — citation graph, markup census, headword-overlap matrix

_Created: 13-07-2026 · Last updated: 23-07-2026_

**What this is.** Q1-WS1.4 of the
[org statistics census & analytics roadmap](../ROADMAP_STATISTICS_ORG_CENSUS_2026_2027.md):
the first FAIR (Findable, Accessible, Interoperable, Reusable) data release for
three already-shipped derived statistics datasets, attacking ATLAS_FAIR gap
**G1** (no persistent identifiers). This document is the deposit package —
metadata, provenance, and license — prepared for a **manual** Zenodo upload.
Executing the actual deposit needs a Zenodo account/API token, which this
session does not have (see [Status](#status) below); a human completes the
upload using the metadata prepared here.

## Why a manual deposit, not a GitHub→Zenodo webhook

[csl-observatory](https://github.com/sanskrit-lexicon/csl-observatory) and
[csl-atlas](https://github.com/sanskrit-lexicon/csl-atlas) are small,
purpose-built repos where a whole-repo GitHub→Zenodo release integration is
appropriate (csl-atlas already carries a prepared
[`.zenodo.json`](https://github.com/sanskrit-lexicon/csl-atlas/blob/main/.zenodo.json)
for exactly that). **This repository is not that shape** — it is a large,
mixed-content research workspace (headword exports, translation drafts,
occasional untracked scan PDFs of uncertain third-party rights). Archiving
the whole repo to Zenodo would sweep in content this repo does not have clear
rights to redistribute. FAIR Release #1 is therefore a **curated, file-level**
deposit: only the two named TSV files below, described by the metadata in
this document, uploaded by hand through the Zenodo web UI (or API once a
token exists).

## What's in the release

| Dataset | File | Rows | Size | Home repo | License |
|---|---|--:|--:|---|---|
| Markup-tag frequency census (E39) | [`data/markup_tag_census.tsv`](markup_tag_census.tsv) | 671 data rows (96 tags × ~7 dicts, per-dict per-tag counts) | 18.5 KB | this repo | CC-BY-4.0 |
| Pairwise headword-overlap matrix (E40) | [`data/headword_overlap_matrix.tsv`](headword_overlap_matrix.tsv) | 105 dictionary pairs + header | 2.9 KB | this repo | CC-BY-4.0 |
| `<ls>` citation-frequency graph (E38) | [`data/citations/`](https://github.com/sanskrit-lexicon/csl-atlas/tree/main/data/citations) (`ls_citation_edges.tsv`, `ls_citation_nodes.tsv`) | 828,505 citations → 912 texts | — | [csl-atlas](https://github.com/sanskrit-lexicon/csl-atlas) | CC-BY-SA-4.0 (csl-atlas's own license) |

E38 is **not duplicated** into this repo — it is cross-referenced by DOI/URL
once minted. This release therefore mints **one concept DOI covering two
files from this repo**, plus a `related_identifiers` link to the citation
graph so the three datasets are discoverable together, without taking on
csl-atlas's licensing or duplicating its data.

## Provenance

- Markup-tag census: generated by [`data/markup_tag_census.py`](markup_tag_census.py)
  over the csl-orig v02 source dictionaries; methodology in
  [`data/MARKUP_TAG_CENSUS_CSLORIG_2026.md`](MARKUP_TAG_CENSUS_CSLORIG_2026.md).
- Headword-overlap matrix: generated by
  [`data/headword_overlap_matrix.py`](headword_overlap_matrix.py) over the
  15-dictionary union headword list; methodology in
  [`data/HEADWORD_OVERLAP_UNION15_2026.md`](HEADWORD_OVERLAP_UNION15_2026.md).
- Citation graph: generated by
  [`build_ls_citation_graph.py`](https://github.com/sanskrit-lexicon/csl-atlas/blob/main/data/citations/build_ls_citation_graph.py)
  in csl-atlas; methodology in
  [`data/citations/README.md`](https://github.com/sanskrit-lexicon/csl-atlas/blob/main/data/citations/README.md)
  there.
