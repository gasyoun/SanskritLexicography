О материале и как пользоваться

Карта слева — позиционная классификация (по А. А. Зализняку). Нажмите на частицу, чтобы открыть разбор справа: позиция, функция, примеры с переводами и ссылками на корпус, отсылки к грамматикам и — где есть — словарную статью Apte (1957). Поиск и фильтры по функциональному типу сужают карту.

Тезис. Неизменяемые слова санскрита, по Whitney §1096, «менее отчётливо разделены на части речи». Это свойство материала, а не дефект описания. При этом, по Gillon (1996), порядок частиц — самый жёсткий элемент синтаксиса: безударные клитики по закону Ваккернагеля встают во 2-ю позицию клаузы (hi, tu — ровно одно слово перед собой; kiṃtu, api tu — ноль). Через частицы виден скелет предложения.

Библиография — полный аппарат

Словари и учебники

  • Apte V. S. The Practical Sanskrit-English Dictionary (1957). Особо подробные статьи: api, eva, hi, tu, iti, khalu, vai.
  • Apte V. S. The Student's Guide to Sanskrit Composition (1885) — корпусный источник для Gillon 1996.
  • Monier-Williams — стандартная сверка.

Грамматика и синтаксис

  • Whitney W. D. Sanskrit Grammar (1879/1924) — §§1096–1135; §1102 — ca; §1104 — api; §1119 — iti.
  • Speyer J. S. Sanskrit Syntax (Leiden, 1886) — §§393–419; §§395–398 — ca, vā; §399 — api; §400 — eva; §402 — hi; §§493–494 — iti.
  • Speyer J. S. Vedische und Sanskrit-Syntax (1896) — для vai, sma, ha.
  • Gillon B. S. Word Order in Classical Sanskrit // Indian Linguistics 57 (1996): 1–35 — закон Ваккернагеля для частиц (§§2, 7).

Гонда — монографические работы

  • Gonda J. The Sanskrit particle api // Lingua.
  • Gonda J. The Indo-European particle *kʷe // Mnemosyne (Leiden), 1954, 177–214; 267–296. Расш.: Vāk 5, Poona, 1957, 1–73.
  • Gonda J. о iva — Vāk / Mnemosyne 1950-х.
  • Gonda J. Stylistic Repetition in the Veda (Amsterdam, 1959) — eva, vai.
  • Gonda J. The Vision of the Vedic Poets (The Hague, 1963) — eva.
  • Hartman C. G. Emphasizing and Connecting Particles in the Thirteen Principal Upanishads (Helsinki, 1966).

Синтаксис и типология

  • Hock H. H. — работы 1980–90-х по синтаксису санскрита, в т. ч. об iti как комплементайзере.
  • König E. The Meaning of Focus Particles (Routledge, 1991) — для eva, api.
  • Güldemann T. Quotative Indexes in African Languages (2008) — для iti в типологии квотативов.
  • Deutscher G. Syntactic Change in Akkadian (2000) — квотатив → комплементайзер.

Русская традиция

  • Зализняк А. А. Древнерусские энклитики (М., 2008) — таблицы рангов и позиционная иерархия.
  • Елизаренкова Т. Я. Грамматика ведийского языка (М., 1982) — vai, ha, sma в ведийском.
