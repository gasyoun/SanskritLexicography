# Syntax-Lectures — санскритские частицы

Учебные материалы (на русском) по санскритским частицам и их синтаксису:
лекционный конспект, визуальные схемы и интерактивный разбор, плюс опорные
переводы словарных статей Apte (1957). Аудитория — первокурсники-индологи,
прошедшие базовую морфологию.

## С чего начать

Главная точка входа — [sanskrit_particles_lectures.md](sanskrit_particles_lectures.md).
Остальные файлы — производные и опорные материалы к нему.

| Файл | Что это |

Линия Whitney — Speyer — Gonda — Apte; позиционная классификация по А. А. Зализняку. |

Собран из конспекта и схемы выше и из серии `Apte_1957-*_RU.md`. |
