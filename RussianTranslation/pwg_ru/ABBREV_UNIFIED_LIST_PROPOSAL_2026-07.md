## Семантика решения (на карточку листа)

- **Принять (approve)** — утвердить предложенное русское соответствие; токен входит в `RU_MAP`.
- **Отклонить (reject)** — оставить как есть (латиница/оригинал + тултип) «по общему согласию» (N5).
- **Отложить (defer)** — обсудить; токен остается как есть до отдельного решения.

Жесткие рамки голосования 19-07-2026: **немецкие токены все переводятся без
исключений** («Русские не читают по-немецки», N3a/N4); латинские грамматические по
умолчанию переводятся (`Caus.` = `кауз.` зафиксировано, N8), «оставить латиницей» —
только явным голосом (N5).

## Система предложений (чтобы список был единым, а не мозаикой)

1. **Падежи** — единый интернационализм: акк., ном., ген., дат., абл., инстр., лок.,
   вок. У санскрита 8 падежей против 6 русских — славянские имена (вин., род.) не
   покрывают аблатив/локатив, поэтому ряд единый транслитерационный.
2. **Времена, наклонения, вторичные основы** — интернационализмы, как у
   зафиксированного «кауз.»: аор., перф., имперф., през., фут., импер., конъ., опт.,
   потенц., прекат., дезид., интенс., деном., пасс., акт., медий.
3. **Универсальные категории** — привычные русские школьные: ед., мн., дв., м., ср. р.,
   прич., инф., сущ., предл., межд., сравн., превосх., перех., неперех., безл.
4. **Контекстные немецкие словосокращения** (`H.` = Herrschaft/Hand, `o. W.` = ohne
   Wissen, `M.` = Mann, `Fr.` = Fremde и т.п.) — фиксированного соответствия НЕТ;
   переводить по расшифровке в атрибуте `n=` конкретного вхождения. Отмечены классом
   «контекстное». Это же ловушка для уже замапленного `W.` («слово»): в store есть
   `W.` с n="Wange" (щека).
5. **Регистрозависимая пара** `Med.`/`med.`: заглавное `Med.` (283 вхождения, в pwgab
   отсутствует) — медий/атманепада, предложение «медий»; строчное `med.` — Medizin,
   остается «мед.». Флаг из ABBREVIATIONS_RU.md разрешается регистром.
