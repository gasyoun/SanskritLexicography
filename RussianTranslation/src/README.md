# `src/` — источники корпусной сверки (этап 4 `pwg_ru`)

Сюда кладутся **оцифрованные санскритско-русские словари**, по которым этап 4
([pwg_ru_prompts/4_korpus_proverka.txt](../pwg_ru_prompts/4_korpus_proverka.txt))
сверяет русский перевод PWG. Шлюз — **неблокирующий аннотатор** с двумя
сигналами; полное описание — в [pwg_ru.md](../pwg_ru.md), раздел 7.

| Файл | Словарь | Роль в шлюзе |
|------|---------|-------------|
| `koch.txt` | Кочергина, Санскритско-русский словарь (современный) | **независимый** авторитет — сигнал **корректности** |
| `kna.txt` | Кнауэр (Knauer), санскритско-русский словарь | **независимый** авторитет — сигнал **корректности** |
| `kow.txt` | Коссович, *Санскрито-русский словарь* (1854); частичный человеческий PWG→рус., посеян из WIL | **эталон** сигнала согласия (сигнал 2) + **вторичное** подтверждение корректности; корректность в одиночку не решает |

Имена файлов выше — **ожидаемое соглашение**; при ином именовании поправьте
ссылки в разделе 7 `pwg_ru.md` и в `4_korpus_proverka.txt`.

**Не кладётся сюда:**

- **FRI** (многоязычный словарь Фриша) — уже на диске в `csl-orig/v02/fri/fri.txt`
  (русская глосса в спанах `<lang n="russian">`); независимый источник корректности.
- **SKD / VCP** — санскритско-санскритские (ноль кириллицы); как русский авторитет
  не используются, только санскритская опора смысла.

Ключ соединения — голый SLP1 (`key1`, с откатом на `key2`), нормализуемый
длиносохраняющей `form_key()`. Покрытие меряется при загрузке **отдельно по
каждому сигналу** (корректность от Кочергина ∪ FRI ∪ KNA; согласие — от KOW).
