# Source Permissions for the Russian Edition

The project has copyright approvals for extensive use of the modern
Sanskrit-Russian reference sources in the publishable `pwg_ru` edition. These
sources are therefore not restricted to evidence-only use.

Covered modern sources currently used by the pipeline:

- Кочергина, `koch`
- Фриш, `fri`
- Смирнов, `smirnov`

Public-domain sources remain usable as before:

- Кнауэр, `kna`
- Коссович, `kow`

Operational rule: preserve source attribution and per-card provenance for
scientific auditability. Do not strip approved modern-source text from the final
card body merely because the source is modern.
