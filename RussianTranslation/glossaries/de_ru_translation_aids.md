## 1. Типы сложных слов (самаса) — русские термины (§7, Apte/Gillon)

| Санскр. тип | Русский термин | Как глоссировать |
|---|---|---|
| tatpuruṣa | татпуруша (определительное/падежное) | по *виграхе*, вершина = **второй** член: *asi-kalaha* → «бой мечом» |
| karmadhāraya | кармадхарая (аппозитивное определительное) | «X, который есть Y» |
| bahuvrīhi | бахуврихи (притяжательное/посессивное) | **экзоцентрично**: *hata-putra-* → «та, у кого убиты сыновья», НЕ сумма членов |
| dvandva | двандва (копулятивное/сочинительное) | «А и Б» |
| avyayībhāva | авьяйибхава (наречное неизменяемое) | наречный оборот |
| dvigu | двигу (числительное определительное) | «совокупность N…» |

**Правила (§7):** строй русскую глоссу от виграхи (правовершинно), не калькируй
по первому члену; член-за-членом калька — повод для пометы. Длинную цепочку
сложных слов разбивай на несколько русских предложений (Apte §356). Сложное на
`-ādi`/`-prabhṛti` = открытый класс = гипероним: «X и тому подобное / и прочее»,
не закрытый список.

## 2. Падежные и абсолютные конструкции (§7, Speyer 1886)

| Конструкция | Русский термин / передача |
|---|---|
| absolute locative | локативный абсолютив (абсолютный локатив) — основная; род. абсолютив — маркированный вариант |
| dativus commodi/incommodi | датив выгоды / заинтересованности |
| dativus finalis | датив цели (взаимозаменяем с инфинитивом) |
| gerund / absolutive | сохраняй деепричастный характер, не уводи в придаточное |
| object of participle/gerund | прямой объект остается в винительном (сильное управление сохраняется при номинализации) |

## 3. Шастрические формулы анализа → фиксированный сухой русский (§7, Tubb)

| Санскр. формула | Русский эквивалент |
|---|---|
| *iti arthaḥ* | то есть; таков смысл |
| *anena … vivakṣitam* | этим он хочет сказать |
| *X-bhāvaḥ* / *X-tvam* | состояние / свойство X |
| *ity ucyate* | называется, именуется |

Наличие этих формул **маркирует** глоссу как шастрический регистр → плоский
терминологический русский, без литературных украшений.

## 4. Коррелятивы yad…tad → русский (§1b/§7, Mitrenina, Zaliznyak-Paducheva, Ruppel, Lipták)

| Относит. (XPrel) | Коррелят (XPana) |
|---|---|
| кто / что | тот / то |
| какой | такой / тот |
| где · куда/откуда | там · туда/оттуда |
| когда · как | тогда · так |
| сколько · чем | столько · тем |
| каков *(арх., только связочн.)* | таков |
| *yo yaḥ* (удвоенный) | кто бы ни / всякий, кто |
| *yatas…tatas* · *yāvat…tāvat* · *yadi…tarhi* | откуда…оттуда · пока…до тех пор · если…то |

**Правила:** препозитивный коррелятив (часто с усилит. **и**: «Кто платит, тот и
заказывает») для генерализующих *yad…tad*; постпозитивный *который* — для
классического типа. Относит. **что** только в им./вин.; падеж **который** — из
его собственного придаточного, никакой аттракции; обязательные корреляты
(тот/то/настолько/тем) не опускать. Множественная вершина → дистрибутив «каждый…
того, с кем…». Умолчание — максимизирующее/универсальное «все, кто…». *yad…tad* —
**слабый** коррелятив: одна форма = рестриктив / аппозитив / условное / косвенный
вопрос; не навязывай одно прочтение (см. рубрику судьи).

## 5. Декодер немецкой орфографии и метаязыка XIX в. (§1c, Delbrück)

Дополняет ГЛОССАРИЙ 2. `Theil→часть`, `thun/That→делать/деяние`, `citirt→цитируется`,
`Uebersetzung→перевод`, `Sprachgebrauch→словоупотребление/узус`, `Stelle→место/пассаж`,
`Wendung→оборот`, `bezeichnet→обозначает`, `Localis→локатив/местный`,
`Instrumentalis→творительный`, `Genera Verbi→залоги глагола`, `erstarrt→застывшая форма`,
`Congruenz→согласование`, `specifisch→специфический`. Латинские пометы
(*Dativus commodi*, *Nomina actionis*) **остаются на латыни**.

## 6. Канонические русские термины (§1a)

- **Падежи:** именительный · родительный · дательный · винительный · творительный · предложный.
- **Семантические роли:** агенс · пациенс · бенефактив(реципиент) · экспериенцер · стимул · инструмент · адресат · источник · цель.
- **Залог/диатеза:** действительный/страдательный залог · возвратный (-ся) · взаимный; каузатив · декаузатив.
- **Управление:** сильное/слабое; беспредложное/предложное.
- Предпочитай канон, не редкое: **агенс** (не агентив), **косвенное дополнение** (не непрямой объект).

## 7. Регистровые пометы (§1d, Apresjan) и пары противоположностей (§6, Riemer)

- **Регистр:** высок./книжн./офиц./поэт. · разг./прост. · груб./жарг./бран. ·
  ласк./неодобр./ирон./презр. · устар./арх./ист./нов. · обл./диал. Регистр русского
  должен соответствовать немецкой помете.
- **Типы противоположности (не смешивать):** комплементарные (*мертвый/живой*) ·
  градуальные (*длинный/короткий*; *не длинный* ≠ *короткий*) · реверсивы
  (*подниматься/опускаться*) · конверсивы (*покупать/продавать*).
- **Маркеры смысловых отношений:** *auch genannt / ein anderes Wort für* → «также
  называется / другое название для»; *nicht…* → «не…» (сохраняй полярность); *Teil von* → «часть…».

## 8. Тесты различения значений — для судьи (§6, Riemer)

Решая, держать ли два немецких подзначения раздельно или слить, применяй три
независимых теста и взвешивай по согласию: (а) **квайновский** (истинно и ложно
об одном референте); (б) **зевгма/тождество** (вынужденное «и … тоже»); (в)
**дефиниционный** (различны, если нет одной максимально-общей-но-минимально-
специфичной глоссы, покрывающей оба). Для метафора/метонимия-связанных значений
требуй *сильного* зевгма-конфликта прежде, чем дробить. **Автогипонимия** (общее +
узкое/ритуальное значение одного слова) — НЕ дублирование, не браковать.
