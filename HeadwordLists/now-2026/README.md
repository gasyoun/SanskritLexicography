# now-2026/ — current-csl-orig regeneration of the headword snapshots

The **NOW (2026)** counterpart to the frozen **THEN (2014)** snapshots in
[`../then-2014/`](../then-2014/). Each file is the sorted-unique headword set of the
dictionary as it stands in the **current**
[`csl-orig/v02`](https://github.com/sanskrit-lexicon/csl-orig/tree/master/v02); the
filename count is the now-count. Regenerated by
[`../headword_diff.py now`](../headword_diff.py); then→now deltas are in
[`../NOW_VS_THEN.md`](../NOW_VS_THEN.md).

## Both keys

- **key1** — the normalised machine key (raw `<k1>`, accent-stripped); use for
  matching / dedup / joins.
- **key2** — the **print / citation form**, re-extracted as clean **SLP1** (keeps the
  `/` udātta accent, `-`/`—` compound markers, `(...)`, `*`, `˚`). The 2014 key2 files
  were in the legacy Cologne numeric transliteration (`am2s4a` = aṃśa); these are the
  modern SLP1 form, so key2 here is the print-ready headword shape (it cannot be
  line-diffed against the numeric 2014 file — that delta is a format migration).

key2 is extracted by stopping the capture at the `¦` headword separator, stripping
`{#..#}` markup, and splitting comma-lists into individual forms (a naïve
`<k2>([^<]*)` over-captures entry body text / compound blobs).

## Caveats

- **Order is normalised Sanskrit collation** (varṇa-krama over SLP1), *not* the
  inconsistent legacy order of the THEN files — compare by **set**
  (`comm`/`sort -u`/the script's set-diff), not a raw line `diff`.
- **PD** is not in csl-orig locally → no NOW file. **PWK** is from csl-orig `pw`
  (Böhtlingk's *kürzere Fassung*).
- Counts drift as csl-orig is corrected; re-run the script to refresh.
