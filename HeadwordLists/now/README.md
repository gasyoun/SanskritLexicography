# now/ — current-csl-orig regeneration of the headword snapshots

These are the **NOW** counterpart to the committed `*-unique-key1-N.txt` snapshots in
the parent directory (the **THEN** versions, frozen at their generation date). Each
file here is the sorted-unique `<k1>` set of the dictionary as it stands in the
**current** [`csl-orig/v02`](https://github.com/sanskrit-lexicon/csl-orig/tree/master/v02);
the filename count is the now-count. Regenerated by
[`../headword_diff.py now`](../headword_diff.py). The parent THEN files are kept
untouched so the two can be compared; the then→now deltas are in
[`../NOW_VS_THEN.md`](../NOW_VS_THEN.md).

## Scope — key1 only, on purpose

Only the **key1** lists are regenerated, because they are the genuinely comparable set:
THEN and NOW are both SLP1 `<k1>`, so a set-diff measures real headword change. The
**key2** lists are *not* regenerated —
- the THEN `<k2>` snapshots are in the **legacy Cologne numeric transliteration**
  (`am2s4a` = aṃśa) while csl-orig is now SLP1 (a format migration, not a headword diff), and
- several dictionaries' raw `<k2>` is not a clean lemma field at all but concatenated
  `{#..#}` compound blobs (SKD, VCP, INM …), so dumping it would be megabytes of markup.

Generated (8): AP, GRA, MW, PWG, PWK, SKD, VCP, VEI. Skipped: PD (not in csl-orig);
all key2; PWK is built from csl-orig `pw` (Böhtlingk's *kürzere Fassung*).

## Caveats

- **Order is normalised Sanskrit collation** (varṇa-krama over SLP1), *not* the
  inconsistent legacy order of the THEN files — compare by **set**
  (`comm`/`sort -u`/the script's set-diff), not a raw line `diff`.
- Counts drift as csl-orig is corrected; re-run the script to refresh.
